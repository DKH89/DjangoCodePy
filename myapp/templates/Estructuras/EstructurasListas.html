Lección 3 : Estructuras de control en Python

7.Listas

En Python, una lista es una estructura de datos versátil que permite almacenar una colección ordenada de elementos. Estas listas pueden contener una variedad de tipos de datos, como números, cadenas, booleanos e incluso otras listas. Gracias a su flexibilidad y facilidad de uso, las listas son uno de los tipos de datos más utilizados en Python. Su capacidad para contener múltiples tipos de elementos y su capacidad para ser modificadas fácilmente las convierten en una herramienta poderosa para almacenar y manipular datos en aplicaciones Python.






7.1 Creación de Listas

Las listas en Python se crean utilizando corchetes y separando los elementos con comas. Esto permite definir fácilmente una lista de elementos, donde cada uno puede ser de cualquier tipo de datos válido en Python. Por ejemplo, una lista de números enteros se vería así: [1, 2, 3, 4, 5], mientras que una lista que contiene una mezcla de tipos de datos, como números, cadenas y booleanos, podría ser [1, "Hola", True, 3.14]. Esta flexibilidad en la creación de listas es una de las características que hacen que las listas sean tan útiles y versátiles en Python.

# Define una lista llamada mi_lista con 5 elementos.

mi_lista =

[

 1 

,

 2 

,

 3 

,

 "Juan" 

]




También se pueden crear listas vacías simplemente utilizando corchetes sin ningún elemento.

# Definición de una lista vacia.

mi_lista =

[ ]




7.2 Acceso a elementos de la lista

Los elementos de una lista en Python se acceden mediante índices, comenzando desde 0 para el primer elemento, 1 para el segundo elemento y así sucesivamente. Esto significa que el primer elemento de la lista tiene un índice de 0, el segundo elemento tiene un índice de 1 y así sucesivamente. Por ejemplo, si tenemos una lista mi_lista con los elementos [10, 20, 30, 40, 50], podemos acceder al primer elemento usando mi_lista[0], al segundo elemento con mi_lista[1], y así sucesivamente.

mi_lista =

[

 1 

,

 2 

,

 3 

,

 "Juan" 

]


print

( mi_lista [ 

0

] )

 # Salida: 1


print

( mi_lista [ 

1

] )

 # Salida: 2




7.3 Metodos en las listas

Los métodos de las listas en Python son funciones integradas que permiten realizar diversas operaciones y manipulaciones en listas. Estos métodos proporcionan una forma conveniente de agregar, eliminar, modificar y buscar elementos en una lista, entre otras acciones. Algunos de los métodos más comunes de las listas en Python incluyen:

append():

Append(): Agrega un elemento al final de la lista.

mi_lista =

[1, 2, 3]


mi_lista.append(4)


print

 (mi_lista) 

# Salida: [1, 2, 3, 4]




extend():

Agrega los elementos de otra lista al final de la lista actual.

mi_lista =

[1, 2, 3]


otra_lista = [4, 5, 6]


mi_lista.extend(otra_lista)


print

 (mi_lista) 

#Salida : [1, 2, 3, 4, 5, 6]

Inserta un elemento en una posición específica de la lista.

mi_lista =

[1, 2, 3]


mi_lista.insert(1, 5)


print

 (mi_lista) 

# Salida: [1, 5, 2, 3]

mi_lista =

print
