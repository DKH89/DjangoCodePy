Lección 3 : Estructuras de control en Python

5.Estructuras Condicionales

Las estructuras condicionales son elementos fundamentales en programación que permiten controlar el flujo de ejecución del código basándose en condiciones específicas. En Python, estas estructuras se definen principalmente con las palabras clave if, elif (opcional) y else. Con estas herramientas, puedes crear programas dinámicos y adaptativos que respondan de manera inteligente a diferentes situaciones, ejecutando diferentes bloques de código según se cumplan o no ciertas condiciones establecidas.




5.1 If Statement

El bloque if es una estructura fundamental en programación que se utiliza para ejecutar un conjunto de instrucciones si una condición dada es verdadera. En Python, este bloque se define con la palabra clave if, seguida de la condición que se evaluará. Si la condición es verdadera, el bloque de código indentado dentro del if se ejecutará; de lo contrario, se omitirá. Esta capacidad de tomar decisiones basadas en condiciones específicas es esencial para la creación de programas que se adapten y respondan de manera inteligente a diferentes situaciones.

La sintaxis básica de un bloque if en Python es la siguiente:


if

condición:


    # Bloque de código que se ejecuta si la condición es verdadera

("El número es positivo")



5.2 Else Statement

El bloque else es una estructura complementaria al bloque if en programación, que se utiliza para ejecutar un conjunto de instrucciones si ninguna de las condiciones anteriores evaluadas con if o elif es verdadera. En Python, el bloque else se define después de todos los bloques if y elif, y su código se ejecutará únicamente si ninguna de las condiciones anteriores se cumple.

La sintaxis básica de un bloque else en Python es la siguiente:

condición:

5.3 Elif Statement

El bloque elif es una estructura condicional adicional que se utiliza en Python para comprobar múltiples condiciones después de evaluar la condición en el bloque if. Si la condición en el bloque if resulta ser falsa, se evalúa la condición en el bloque elif. Esto permite crear una lógica más compleja y detallada en la toma de decisiones en un programa. Si la condición en el bloque elif es verdadera, se ejecuta el bloque de código asociado a ese elif.

La sintaxis básica de un bloque elif en Python es la siguiente:

elif
