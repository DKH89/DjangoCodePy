Lección 3 : Estructuras de control en Python

9.Diccionarios

En Python, los diccionarios son estructuras de datos fundamentales que permiten almacenar pares clave-valor. A diferencia de las listas, que se indexan mediante números enteros, los diccionarios se indexan mediante claves, las cuales pueden ser de cualquier tipo inmutable, como cadenas, números o tuplas. Debido a esta capacidad de indexación personalizada, los diccionarios son altamente versátiles y se utilizan ampliamente en Python. Su capacidad para almacenar y recuperar datos de manera eficiente los convierte en una herramienta indispensable para el manejo y organización de datos en diversos tipos de aplicaciones.





9.1 Creación del diccionario

Los diccionarios en Python se crean utilizando llaves {} y especificando los pares clave-valor separados por comas. Esta estructura permite una representación clara y concisa de la relación entre las claves y sus respectivos valores. Por ejemplo, podemos definir un diccionario que represente información sobre una persona con claves como "nombre", "edad" y "ciudad", y asignar los valores correspondientes a cada una de ellas.

# Definición de un diccionario llamado mi_diccionario con tres pares clave-valor.

mi_diccionario =

{

"nombre"

:

"Juan"

,

"edad"

:

30

,

"ciudad"

:

"Madrid"

}



9.2 Acceso a Elementos de un diccionario

Los elementos de un diccionario en Python se acceden mediante sus claves. Cada elemento en un diccionario está asociado a una clave única, y esta clave se utiliza para recuperar el valor correspondiente almacenado en el diccionario. Por ejemplo, si tenemos un diccionario que contiene información sobre una persona con claves como "nombre", "edad" y "ciudad", podemos acceder al valor del nombre utilizando la clave "nombre".

print

( mi_diccionario [ 

0

] )

 # Salida: Juan


print

( mi_diccionario [ 

1

] )

 # Salida: 30




Si se intenta acceder a una clave que no está presente en el diccionario, se producirá un error KeyError. Para evitar este error, se puede utilizar el método get(), que devuelve None si la clave no está presente:

print

] )

 # Salida: none




9.3 Modificación de Elementos del Diccionario

Los elementos de un diccionario en Python pueden modificarse de manera dinámica asignando nuevos valores a las claves existentes o añadiendo nuevos pares clave-valor según sea necesario. Esto proporciona una gran flexibilidad en la manipulación de datos dentro del diccionario. Por ejemplo, si queremos actualizar la edad de una persona en un diccionario que contiene información sobre ella, simplemente podemos asignar un nuevo valor a la clave "edad". Del mismo modo, si queremos añadir información adicional, como el apellido de la persona, podemos hacerlo fácilmente añadiendo un nuevo par clave-valor al diccionario.

mi_diccionario[ 

"edad"

 ] =

35

    # Modificar el valor de la clave "edad"


mi_diccionario[ 

"apellido"

 ] =

"García"

    # Añadir un nuevo par clave-valor
