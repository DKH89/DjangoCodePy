Lección 3 : Estructuras de control en Python

6.Estructuras Ciclicas

Las estructuras cíclicas, conocidas comúnmente como bucles, son esenciales en la programación para repetir la ejecución de un conjunto de instrucciones mientras ciertas condiciones se mantengan válidas. En Python, estas estructuras se implementan principalmente a través de los bucles for y while, que ofrecen distintas formas de controlar la repetición de código en función de la situación. Estos bucles son fundamentales para automatizar tareas repetitivas y para procesar colecciones de datos de manera eficiente, proporcionando así una herramienta poderosa para el desarrollo de programas en Python.




6.1 Bucle For

El bucle for es una estructura en Python que permite iterar sobre una secuencia de elementos, como una lista o una tupla, y ejecutar un bloque de código para cada elemento en esa secuencia. Esta estructura proporciona una forma eficiente y elegante de procesar colecciones de datos, ya que el bucle itera automáticamente sobre cada elemento en la secuencia, permitiendo realizar operaciones específicas en cada uno de ellos. El bucle for es especialmente útil cuando se necesita realizar la misma operación en todos los elementos de una colección, lo que lo convierte en una herramienta esencial en el desarrollo de programas en Python.

6.2 Bucle While

El bucle while es una estructura de control en Python que se emplea para ejecutar un bloque de código repetidamente mientras una condición específica se mantenga verdadera. Esto significa que el bloque de código dentro del bucle se ejecutará una y otra vez hasta que la condición se vuelva falsa. Es importante tener en cuenta que la condición se evalúa antes de cada ejecución del bloque de código, por lo que si la condición ya es falsa al inicio, el bloque de código no se ejecutará ni una sola vez. El bucle while es útil cuando no se sabe cuántas veces se debe ejecutar el bloque de código de antemano, sino que se basa en la evaluación continua de una condición.

while

condicion:


    

# Bloque de código que se ejecuta mientras la condición sea verdadera
