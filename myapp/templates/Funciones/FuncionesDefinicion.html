Lección 4 : Funciones en Python

10. Definición de Funciones

Las funciones en Python son bloques de código reutilizables que realizan una tarea específica cuando son llamadas. Proporcionan una forma de estructurar y modularizar el código, lo que facilita su mantenimiento y comprensión. Al definir una función, se especifica el nombre de la función y el conjunto de instrucciones que se ejecutarán cuando la función sea llamada. Las funciones pueden aceptar cero o más argumentos, y pueden devolver un valor opcionalmente.






10.1 Sintaxis de una función

En Python, la definición de una función sigue una sintaxis clara y estructurada. Se inicia con la palabra clave def, seguida del nombre de la función y entre paréntesis los parámetros que puede recibir. Luego, se utiliza dos puntos : para indicar el comienzo del cuerpo de la función, que está indentado. En este cuerpo se encuentran las instrucciones que definen las acciones que la función realizará cuando sea llamada. Opcionalmente, la función puede contener una declaración return para devolver un valor específico al lugar desde donde fue llamada.

    # Puede devolver un valor utilizando la palabra clave return



10.2 ¿Como llamamos a una función?

Llamar a una función en Python significa invocarla para que ejecute su código. Esto se hace escribiendo el nombre de la función seguido de paréntesis, y proporcionando los argumentos necesarios si la función los requiere. Por ejemplo, si tenemos una función llamada saludar que acepta un nombre como argumento, la llamamos simplemente escribiendo saludar("Juan"). Esto le indica a Python que ejecute el código dentro de la función saludar, utilizando "Juan" como valor para el parámetro nombre.
