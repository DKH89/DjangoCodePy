3. Variables en Python

En Python, las variables son contenedores utilizados para almacenar datos. Lo que distingue a Python de otros lenguajes de programación es que no es necesario declarar explícitamente el tipo de variable antes de asignarle un valor. En lugar de eso, el intérprete de Python deduce automáticamente el tipo de datos basándose en el valor asignado.





3.1 Declaración de Variables

Para declarar una variable en Python, simplemente se asigna un valor a un nombre utilizando el operador de asignación =. Esta sintaxis es directa y fácil de entender, lo que facilita la creación y asignación de variables en el código. Por ejemplo, si queremos asignar el valor "Hola" a una variable llamada mensaje, simplemente escribimos mensaje = "Hola"


# Declara la variable 'nombre' con el valor "Juan" y 'edad' con el valor 25.


nombre =

"Juan"


edad =

15





3.2 Tipos de Variables en Python

● Enteros (int): Los enteros en Python, representados por el tipo de datos int, son números enteros sin punto decimal. Estos números pueden ser positivos o negativos, y pueden ser utilizados para representar cantidades completas sin fracciones. Por ejemplo, 0, 1, -5, y 1000 son todos ejemplos de enteros en Python.

● Flotantes (float): Los flotantes en Python, representados por el tipo de datos float, son números con punto decimal. Estos números pueden representar cantidades que incluyen valores fraccionarios o números decimales. Por ejemplo, 3.14, -0.5, y 2.0 son ejemplos de flotantes en Python.

● Cadenas (str): Las cadenas en Python, representadas por el tipo de datos str, son secuencias de caracteres. Estas secuencias pueden contener letras, números, y otros caracteres especiales.

● Booleanos (bool): Los booleanos en Python, representados por el tipo de datos bool, son valores que pueden ser True (verdadero) o False (falso). Estos valores son utilizados para representar el resultado de comparaciones o condiciones lógicas en un programa Python.

3.3 Convenciones de Nombres

Es importante seguir algunas convenciones al nombrar variables en Python. Por lo general, los nombres de las variables son en minúsculas, con palabras separadas por guiones bajos para mejorar la legibilidad. Por ejemplo:

# Forma correcta de nombrar variables en Python


nombre_completo =

"Maria Gonzalez"


edad_usuario =

30




3.31 Datos a tener en cuenta.


● Los nombres de variables pueden contener letras, números y guiones bajos.

● Los nombres de variables no pueden comenzar con un número.

● Python distingue entre mayúsculas y minúsculas, por lo que Nombre y nombre serían variables diferentes.
