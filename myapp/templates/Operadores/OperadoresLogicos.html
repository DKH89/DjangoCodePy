Lección 6 : Operadores en Python

21.Operadores Logicos

Los operadores lógicos en Python desempeñan un papel fundamental al combinar expresiones lógicas para evaluar condiciones más complejas. Al aplicar estos operadores, es posible realizar operaciones booleanas sobre valores booleanos, y el resultado de estas operaciones también es un valor booleano. Estos operadores, que incluyen AND, OR y NOT, son esenciales para la construcción de estructuras de control de flujo y para la toma de decisiones en Python. Permiten la creación de condiciones compuestas que dependen de múltiples variables o estados, lo que proporciona una forma poderosa de expresar la lógica en programas Python.








21.1 Operador AND

El operador lógico and en Python se utiliza para combinar dos expresiones booleanas y evaluar si ambas son verdaderas. Devuelve True solo si ambas expresiones son verdaderas; de lo contrario, devuelve False. En otras palabras, el operador and requiere que todas las condiciones vinculadas sean verdaderas para que la expresión completa sea verdadera. Este operador es fundamental en la construcción de condiciones compuestas, donde se necesita que múltiples condiciones sean verdaderas para que se ejecute cierto bloque de código.

21.2 Operador OR

El operador lógico or en Python se utiliza para combinar dos expresiones booleanas y evaluar si al menos una de ellas es verdadera. Devuelve True si al menos una de las expresiones es verdadera; si ambas expresiones son falsas, devuelve False. En otras palabras, el operador or requiere que al menos una de las condiciones vinculadas sea verdadera para que la expresión completa sea verdadera. Este operador es esencial para construir condiciones en las que se requiere que al menos una de varias condiciones sea verdadera para ejecutar un bloque de código específico.

21.3 Operador Not

El operador lógico not en Python se utiliza para negar una expresión booleana, es decir, cambia True a False y False a True. Este operador es útil para invertir el valor de una expresión booleana. Por ejemplo, si se tiene una condición que se evalúa como verdadera, al aplicar el operador not, la condición se volverá falsa y viceversa. El operador not es especialmente útil cuando se desea verificar si algo no es cierto o no está presente. Por ejemplo:
