Lección 6 : Operadores en Python

19. Operadores de Aritmeticos

Los operadores aritméticos en Python son fundamentales para la realización de operaciones matemáticas en valores numéricos. Estos operadores estándar permiten realizar una variedad de cálculos, incluyendo sumas, restas, multiplicaciones, divisiones y otras operaciones básicas. Al utilizar estos operadores, los programadores pueden llevar a cabo diversas tareas matemáticas de manera eficiente y sencilla en sus programas Python. Esta funcionalidad es crucial en una amplia gama de aplicaciones, desde el procesamiento de datos hasta la simulación de sistemas físicos, y brinda a los desarrolladores la flexibilidad necesaria para realizar operaciones complejas con facilidad.








19.1 Tipos de Operadores Aritmeticos

Los operadores aritméticos en Python se utilizan para realizar diversas operaciones matemáticas.

● Suma (+) : Se utiliza para sumar dos valores.

# La variable X equivale a la suma de 5 más 6


x = 5 + 6




● Resta (-) : Se utiliza para restar dos valores.

# La variable X equivale a la resta de 5 menos 6


x = 5 - 6




● Multiplicación (*) : Se utiliza para multiplicar dos valores.

# La variable X equivale a la multiplicación de 5 por 6


x = 5 * 6




● División (*) : Se utiliza para dividir un valor por otro. El resultado siempre es un número decimal, incluso si ambos operandos son enteros.

# La variable X equivale a la división de 5 dividio en 6


x = 5 / 6




● Módulo (%) : Se utiliza para obtener el resto de la división entre dos valores.

# La variable X equivale a la modulo de la división entre 5 y 6


x = 5 % 6




● Exponente (**) : Se utiliza para elevar un valor a la potencia de otro.

# La variable X equivale a 5 elevado a la 6


x = 5 ** 6
