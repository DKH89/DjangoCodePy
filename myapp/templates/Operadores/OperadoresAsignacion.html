Lección 6 : Operadores en Python

18.Operadores de Asignación

Los operadores de asignación en Python son esenciales para asignar valores a variables. Además de la asignación simple, donde un valor se asigna directamente a una variable mediante el operador =, Python proporciona operadores de asignación compuesta. Estos operadores permiten combinar una operación aritmética con la asignación, lo que proporciona una forma concisa y legible de actualizar el valor de una variable en función de su valor actual y otro valor. Esta funcionalidad no solo simplifica el código, sino que también puede mejorar la eficiencia al evitar la duplicación de cálculos innecesarios.








18.1 Operador de asignación simple (=)

Se utiliza para asignar un valor a una variable.

# Se asigna a la variable x el valor de 10


x =

10




18.2 Operadores de asignación compuesta

Estos operadores combinan una operación aritmética con la asignación. Algunos ejemplos comunes incluyen:

● "+=" : Para sumar y asignar.

# Equivalente a: x = x + 5


x +=

5




● "-=" : Para restar y asignar.

# Equivalente a: y = y - 3


y -=

3




● "*=" : Para multiplicar y asignar.

# Equivalente a: z = z * 2


z *=

2




● "/=" : Para dividir y asignar.

# Equivalente a: w = w / 4


w /=

4
